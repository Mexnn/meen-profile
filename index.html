Get to know me!

ปัจจุบันผมกำลังศึกษาในระดับ ปวส. แผนกคอมพิวเตอร์ธุรกิจ ที่วิทยาลัยเทคโนโลยีไออาร์พีซี (IRPC) ครับ ซึ่งการเรียนที่นี่ทำให้ผมได้ฝึกฝนทักษะด้านไอทีและการจัดการธุรกิจควบคู่กันไป ผมมีพื้นฐานการเขียนโค้ดและมีความสามารถในการจัดการระบบคอมพิวเตอร์ ทั้งการประกอบ แก้ไขปัญหา และการติดตั้งซอฟต์แวร์ต่างๆ ผมเป็นคนที่ชอบเรียนรู้เทคโนโลยีใหม่ๆ อยู่เสมอ และพร้อมที่จะนำความรู้จากห้องเรียนมาประยุกต์ใช้กับการทำงานจริงครับ

โปรแกรมจัดไฟล์ในเครื่อง

โปรแกรมจัดการและจัดระเบียบไฟล์ที่ผมพัฒนาขึ้นด้วยภาษา Python ทั้งหมด โดยออกแบบมาเพื่อช่วยแก้ปัญหาไฟล์กระจัดกระจายในเครื่องคอมพิวเตอร์ โปรแกรมนี้มาพร้อมกับหน้าจอใช้งาน (UI) ที่ทันสมัยและเข้าใจง่าย สามารถแยกประเภทไฟล์ลงโฟลเดอร์ต่างๆ ได้อัตโนมัติ ช่วยลดเวลาในการจัดระเบียบข้อมูลได้อย่างมีประสิทธิภาพครับ

รับชมผลงาน

เว็ปเก็บกิจกรรมกยศ.

ระบบเว็บแอปพลิเคชันสำหรับจัดการกิจกรรมจิตอาสาเพื่อประกอบการยื่นกองทุนกยศ. พัฒนาด้วยภาษา PHP และจัดการฐานข้อมูลด้วย MySQL อย่างเต็มรูปแบบ ระบบนี้รองรับการทำงานตั้งแต่การสมัครสมาชิกและเข้าสู่ระบบที่ปลอดภัย มีฟังก์ชันการอัปโหลดไฟล์เอกสาร PDF เพื่อยืนยันการทำกิจกรรม รวมถึงสามารถดาวน์โหลดข้อมูลกลับมาเพื่อตรวจสอบได้ ช่วยลดความซับซ้อนในการเก็บหลักฐานและทำให้การจัดการชั่วโมงจิตอาสาสะดวกยิ่งขึ้นครับ
